ZENTRYPAY - PLESK DEPLOYMENT INSTRUCTIONS
==========================================

UPLOAD THESE FILES TO YOUR PLESK:
1. Upload ALL files from this folder to: /httpdocs/app/
2. The structure should be:
   /httpdocs/app/
   ├── server/
   │   └── node-build.mjs
   ├── spa/
   │   ├── index.html
   │   └── assets/
   ├── package.json
   └── .env.example

PLESK CONFIGURATION:
- Application Root: /httpdocs/app
- Startup File: server/node-build.mjs
- Node.js Version: 18.x or 20.x
- Environment Variables:
  * NODE_ENV = production
  * PORT = 3000

PROXY SETTINGS (Apache/nginx):
ProxyPass / http://localhost:3000/
ProxyPassReverse / http://localhost:3000/
ProxyPreserveHost On

AFTER UPLOAD:
1. Install dependencies (NPM Install in Plesk)
2. Enable Node.js application
3. Configure proxy settings
4. Test: yourdomain.com should show login page
